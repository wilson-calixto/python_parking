## API de CRUD com Flask
Uma pequena API de CRUD com Python e Flask.

## Como Executar o Projeto

```bash
~$ export FLASK_APP=app_parking
~$ export FLASK_ENV=Development
~$ export FLASK_DEBUG=True
~$ flask run	
```

## Como realizar as Migrações

```bash
~$ flask db init
~$ flask db migrate
~$ flask db upgrade
```
